What Is a To-Do List?
•	A to-do list helps us remember our work
•	We can write things we need to do
•	We can mark work as done
•	We can remove work we don’t need
________________________________________
What Can This App Do?
•	Add new tasks 
•	Mark tasks as finished 
•	Edit tasks 
•	Delete tasks 
•	Move tasks up and down 
________________________________________

Three Parts of the App
This app is made using 3 things:
•	HTML – Makes the page
•	CSS – Makes the page look nice
•	JavaScript – Makes the page work
________________________________________

1.	HTML (The Body)
HTML creates:
•	A title (To-Do List)
•	A box in the center
•	A text box to type tasks
•	An Add button
•	A list to show tasks
HTML decides what appears on the page
________________________________________


2.	CSS (The Design)
CSS makes the app beautiful:
•	Adds a background image
•	Colors the task box
•	Makes rounded corners
•	Adds shadow
•	Changes task color when done
CSS decides how the page looks
________________________________________

3.	JavaScript (The Brain)
JavaScript makes the app smart:
•	Adds tasks
•	Deletes tasks
•	Edits tasks
•	Marks tasks as done
•	Moves tasks
•	Saves tasks
JavaScript decides what happens when we click
________________________________________
 
•	Adding a Task
Steps:
1.	Type a task
2.	Click the Add button
3.	Task appears in the list

________________________________________

•	Completing a Task
Steps:
1.	Click the checkbox
2.	Task gets a line
3.	Task is finished

________________________________________
•	Editing a Task
Steps:
1.	Click Edit
2.	Change the task text
3.	Click OK
This helps fix mistakes ✏️
________________________________________
•	Deleting a Task
Steps:
1.	Click Delete
2.	Task is removed
This helps keep the list clean
________________________________________
•	Moving Tasks
•	Click and drag a task
•	Move it up or down
•	Drop it in the right place
This helps arrange tasks in order
